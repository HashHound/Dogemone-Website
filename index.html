About Dogemone

Dogemone is a decentralized, privacy-focused cryptocurrency built on the CryptoNote protocol. Our mission is to provide high levels of security, privacy, and decentralization to users around the globe.

Features

Privacy

Utilizes ring signatures and stealth addresses to obfuscate transaction details, ensuring user privacy.

Security

Based on the robust CryptoNote protocol, providing secure and reliable transactions.

Decentralization

Fully decentralized network with no central authority.

Mining Algorithm

Uses the CryptoNight algorithm for proof-of-work mining.

Community-Driven

We are an open-source project. Anyone can contribute.

No Halving/Emission Curve

We Offer a Fixed Block Reward for the Fair Distribution of Coins, Dogemone does not have a Halving Schedule.

How to Mine Dogemone

Follow these steps to start mining Dogemone:

  1. Download the Wallet and Node: Get the latest version of the Dogemone Release from our releases page.
  2. Generate an Address: Use the simplewallet to generate a new wallet address.
  3. Run a Node: Start the Dogemone daemon with ./dogemoned --rpc-bind-ip 0.0.0.0 --rpc-bind-port 53000
  4. Start Mining: Once the wallet gets connected to the node, type start_mining on the wallet. Mining will be started and you will recieve DME in this Wallet.

Dogemone (DME) Details

  • Algorithm: Cryptonight
  • Coin name: Dogemone
  • Coin abbreviation: DME
  • Block reward: 57.6 coins
  • Tail reward: 0.3 coins
  • Coin supply: 300,000,000 coins
  • Block Time: 60 seconds
  • Halving: No Halving
  • Difficulty Adjustment: Dogemone Difficulty Adjustment
  • Dev Fee: Every 10th block which is roughly 10%
  • RPC port: 53000
  • P2P port: 49200
